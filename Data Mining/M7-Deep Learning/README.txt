# Employee Attrition Prediction

This project focuses on predicting employee attrition using various machine learning models. It includes data preprocessing, exploratory analysis, and the implementation of models like logistic regression, decision trees, and random forests.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation

## Performance Metrics
- Accuracy
- Precision
- Recall
- ROC AUC

## Usage
1. Clone the repository
2. Navigate to the `Employee_Attrition_Prediction` folder
3. Open the `Employee_Attrition_Prediction.ipynb` notebook
4. Run the cells to see the analysis
