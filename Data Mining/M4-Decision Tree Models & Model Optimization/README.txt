# Wine Quality Prediction

This project aims to predict wine quality based on physicochemical properties. It includes data preprocessing, exploratory data analysis, and model building using k-NN.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation

## Performance Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC AUC

## Usage
1. Clone the repository
2. Navigate to the `Wine_Quality_Prediction` folder
3. Open the `Wine_Quality_Prediction.ipynb` notebook
4. Run the cells to see the analysis
