# Initial Analysis - Air Quality Prediction

This project involves predicting air quality based on pollutant concentrations using logistic regression. It includes data preprocessing, exploratory data analysis, and model evaluation.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation

## Performance Metrics
- Confusion matrix
- Classification report
- ROC curve

## Usage
1. Clone the repository
2. Navigate to the `Air_Quality_Prediction` folder
3. Open the `Air_Quality_Prediction.ipynb` notebook
4. Run the cells to see the analysis
