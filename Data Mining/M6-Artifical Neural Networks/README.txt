# Marketing Campaign Response Prediction

This project predicts the response to a marketing campaign using machine learning models. It involves data cleaning, exploratory data analysis, and the application of logistic regression.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation

## Performance Metrics
- Confusion matrix
- Classification report
- ROC curve

## Usage
1. Clone the repository
2. Navigate to the `Marketing_Campaign_Response_Prediction` folder
3. Open the `Marketing_Campaign_Response_Prediction.ipynb` notebook
4. Run the cells to see the analysis
