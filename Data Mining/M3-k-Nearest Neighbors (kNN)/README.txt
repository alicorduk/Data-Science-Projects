# Comparing Model Performance

This project compares the performance of different machine learning models on a dataset. Models evaluated include logistic regression, k-NN, decision trees, random forests, and Naive Bayes.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation
- Comparison of model performance

## Performance Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC AUC

## Usage
1. Clone the repository
2. Navigate to the `Comparing_Model_Performance` folder
3. Open the `Comparing_Model_Performance.ipynb` notebook
4. Run the cells to see the analysis
