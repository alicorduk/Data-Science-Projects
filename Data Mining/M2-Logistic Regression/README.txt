# Diabetes Prediction

This project involves predicting diabetes in patients using various features. It includes data preprocessing, exploratory data analysis, and model building using logistic regression.

## Contents
- Data preprocessing
- Exploratory data analysis
- Model building and evaluation

## Performance Metrics
- Confusion matrix
- Classification report
- ROC curve

## Usage
1. Clone the repository
2. Navigate to the `Diabetes_Prediction` folder
3. Open the `Diabetes_Prediction.ipynb` notebook
4. Run the cells to see the analysis
