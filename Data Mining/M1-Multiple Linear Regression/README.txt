# Real Estate Price Prediction

This project predicts real estate prices using multiple linear regression. It includes data cleaning, exploratory analysis, and regression modeling.

## Contents
- Data preprocessing
- Exploratory data analysis
- Regression modeling

## Performance Metrics
- R-squared
- MAE
- RMSE
- Max error

## Usage
1. Clone the repository
2. Navigate to the `Real_Estate_Price_Prediction` folder
3. Open the `Real_Estate_Price_Prediction.ipynb` notebook
4. Run the cells to see the analysis
